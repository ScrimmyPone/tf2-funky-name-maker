## Features:
- Ability to generate cool funky names using a variety of special characters and symbols
- Ability to select how many variations you'd like
- Ability to restrict the character sets used
- Ability to choose whether to add random symbols before/after the name, and whether to space them out
- Symbol picker at bottom of page that allows you to select a character set, shows the characters/symbols in a neatly organized grid system, and you can copy symbols by clicking on them

**DISCLAIMER:** Partially created using Claude Sonnet 4.6 (AI / LLM) for code generation. **I DO NOT support the use of AI for artwork, deepfakes, or other scummy uses**, please use AI responsibly.

## How to use this:

The easiest option is to simply use the public website hosted by Github at: [https://scrimmypone.github.io/tf2-funky-name-maker/](https://scrimmypone.github.io/tf2-funky-name-maker/)

Alternatively, simply download the `index.html` file and open it up in your preferred web browser (Firefox, Chromium, Safari, whatever you want. Only tested it in Firefox so far) - or download the ZIP file from the releases page, extract it, and then open the `index.html` file from there.
