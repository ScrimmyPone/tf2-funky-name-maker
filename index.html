What is this and how do I use it?

This simple web tool is intended for Name Tags and Description Tags in the video game Team Fortress 2 made by Valve.

At the bottom of the page, there's a Symbol Picker (similar to Windows' Character Map), where you can pick a character set and it'll show you all the characters and symbols available from the set (at the bottom of the list there's "All Symbols Combined" to see all the random symbols you can use - note the All Symbols group does not include the A-Z / 0-9 character variations, just things like smiley faces, certain japanese characters, various shapes and symbols, etc.).

PLEASE BE AWARE: TF2 may render some symbols on this page differently than shown in your browser, and certain symbols may appear cut-off in-game. Please pay close attention to the preview in-game before confirming your name/description tag usage, I am not responsible if you waste a name tag because some of the symbols don't show up properly in-game.

⚙ Weapon Name

Simply enter a name such as "My Cool Knife", adjust the "Variants" to choose how many random versions of the name you'd like to generate, then click the "Forge Name" button, and you'll find a bunch of cool variants of your name such as "❋❊ ＭＹ ＣＯＯＬ ⓀⓃⒾⒻⒺ ✰" below, which you can simply click "Copy" on the right to copy it to your clipboard for pasting into a TF2 Name Tag / Description Tag.

⚙ Character Set Mode

Randomly picks 1–2 character sets each time you generate.

There's various other settings, for example you can manually pick which character set(s) to use, or you can change whether or not to add random symbols before/after the generated namex.

⚙ Symbol Decorations

1–2 random symbols will be added to the start and end of the name.

⚿ Character & Symbol Browser

Once you've selected a character set, simply scroll down and click on the symbol you want to use, it'll copy it to your clipboard which you can then paste into a text editor like Notepad (Windows) / TextEdit (macOS) while you're editing your generated name to include your preferred symbols and spacing.

You can adjust the font size and number of columns to make it easier to see the symbols if you're having trouble figuring out what they're supposed to be.
